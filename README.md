# Lava Rebalance

Changes some features of lava to prevent it from being as easily renewable.
All features of the mod can be individually toggled in ``minecraft/config/lavarebalance.cfg``.

- Prevents any form of cobble from being generated. 
- Prevents lava transmuting redstone dust on touch.
- Prevents lava melting igneous cobbled netherrack on touch.

These changes should work just fine on a server without the need for it to also be installed on the client.
